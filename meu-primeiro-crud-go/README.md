# Meu Primeiro CRUD em Go

Essa é a sessão que vai conter os códigos do curso:

-  **[Meu Primeiro CRUD em Go do HunCoding](https://www.youtube.com/playlist?list=PLm-xZWCprwYQ3gyCxJ8TR1L2ZnUOPvOpr)**

O objetivo desse curso é ensinar o básico para criar uma API simples, um CRUD de usuários.

Ele vai utilizar uma [arquitetura MVC](https://www.devmedia.com.br/introducao-ao-padrao-mvc/29308), banco de dados [MongoDB](https://www.mongodb.com/) e o [Gin Gonic](https://gin-gonic.com/) para framework HTTP. Também vai conter validação de token JWT, validação dos dados de request e separação de responsabilidades em camadas.

É um projeto básico para eu aprender sobre criação de API's em Go.
